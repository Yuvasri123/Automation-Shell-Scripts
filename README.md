# Linux Automation Shell Scripts

## Overview
This repository contains Bash shell scripts developed to automate Linux system administration, monitoring, user management, and operational tasks across multiple servers.

## Technologies
- Bash / Shell Scripting
- Linux (RHEL / Ubuntu)
- System Administration
- Automation

## Scripts Included

### System Monitoring & Performance
- Health_Check_Multiple_Linux_serverScript.sh
- weekly_CPU_RAM_utilization_Script.sh
- unix_metrics_for_mmi_server_script.sh

### User & Access Management
- Password_reset_Script_Update.sh
- TempAdminRevokeValidation.sh
- pre_access.sh
- post_access.sh

### System Configuration
- kernal_Parameter_Configuration_Script.sh
- FileSystemExtension.sh

### Backup & Storage Operations
- NetBackup_script.sh
- 3PAR_Remote_Copy_Group.sh

### Miscellaneous
- SHOWVL~1.SH

## Key Features
- Automation of routine system administration tasks  
- Multi-server health monitoring  
- User access and privilege management  
- Performance and resource utilization tracking  
- Backup and storage operation support
